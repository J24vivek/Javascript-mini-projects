# JavaScript Concept-Based Mini Projects

This repository contains multiple mini projects built using **HTML, CSS, and JavaScript**.  
The goal of these projects is to strengthen core JavaScript concepts by building real-world features without using any frameworks.

---

## Key Learnings
- DOM manipulation and event handling
- Writing clean and reusable JavaScript logic
- Improving debugging and problem-solving skills
- Building interactive applications without libraries
